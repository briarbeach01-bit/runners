4-Week Beginner Plan (3 runs/week)

Goal: build a habit, avoid injury, finish each run feeling “okay.” If you’re brand new, use run/walk (example: run 2 min, walk 1 min).

Week 1
Run A
20–25 min easy (run/walk if needed)
Run B
20–25 min easy
Run C
25–30 min easy
Week 2
Run A
25 min easy
Run B
25 min easy
Run C
30–35 min easy
Week 3
Run A
25–30 min easy
Run B
25–30 min easy + 4 × 20 sec “faster” (optional)
Run C
35–40 min easy
Week 4
Run A
30 min easy
Run B
30 min easy + 6 × 20 sec “faster” (optional)
Run C
40–45 min easy

Optional on non-run days: 10–20 minutes brisk walking, light strength, or mobility.
